ngPick

Parâmetros

Parâmetro Default Descrição
input Variavel de entrada, contendo uma lista
output Variavel de saída
var Vazio Variavel para ser utilizado na declaração das colunas
input-title Vazio Título da listagem de entrada
output-title Vazio Título da listagem de saída
list-height 200px Altura das listagens, utilizando valores px, %, ...
list-width 240px Largura das listagens, utilizando valores px, %, ...
add-all-label Adicionar Todos Label do botão de adicionar todos os items
remove-all-label Remover Todos Label do botão de remover todos os items
add-label Adicionar Item Label do botão de adicionar item
remove-label Remover Item Label do botão de remover item
first-label Primeiro Label do botão de deslocar item para a primeira posição
last-label Último Label do botão de deslocar item para a última posição
up-label Sobe Label do botão de deslocar item para cima
down-label Desce Label do botão de deslocar item para baixo
order-control-visible visivel Opção de visualizar menu de ordenação de items

Modo Simples

Utilizando arrays de String.

<ng-pick input="exemplo_entrada_1" output="exemplo_saida_1" input-title="Entrada" output-title="Saída">
</ng-pick>

Column

Parâmetro Default Descrição
caption Vazio Título da coluna
width Largura da coluna, utilizando valores px, %, ...

Array de Objetos

Para listar dados de objetos será preciso utilizar o parâmetro var e a tag column.

{{linha.name}} 
<ng-pick input="entrada" output="saida" input-title="JSONs de Entrada" output-title="JSONs de Saída" list-width="300px" var="linha" order-control-visible="false">
    <column caption="Name">{{linha.name}}</column>
</ng-pick>

Diretivas

Através da tag column é possível utilizar diretivas do AngularJS

{{entrada2}}
{{saida2}} 
<ng-pick input="entrada2" output="saida2" input-title="Diretivas de Entrada" output-title="Diretivas de Saída" list-width="300px" var="linha" add-all-label=">>" remove-all-label="<<" add-label=">" remove-label="<">
    <column caption="" width="40px"><span class="glyphicon" ng-class="linha.value"></span></column>
   <column caption="Valor"><input type="text" name="input" ng-model="linha.name"></input></column>
</ng-pick>
