ngGrid

Parâmetros

Parâmetro Default Descrição
input Variavel de entrada, contendo uma lista
output Variavel de saída
var Vazio Variavel para ser utilizado na declaração das colunas
title Vazio Título da Grid
rows Vazio Quantidade de linhas exibidas.

Column

Parâmetro Default Descrição
caption Vazio Título da coluna
width Largura da coluna, utilizando valores px, %, ...
order Ordenação da coluna
filter Filtro da coluna

Modo Simples

Listagem Simples, sem paginação e seleção.

{{linha.id}} {{linha.name}} {{linha.value}} 
<ng-grid input="entrada1" var="linha">
  <column caption="ID" width="100px">{{linha.id}}</column>
  <column caption="Name" order="name" filter="name" width="200px">{{linha.name}}</column>
  <column caption="Value" order="value">{{linha.value}}</column>
</ng-grid>

Paginação e seleção

Listagem com paginação e seleção.

{{linha.id}} {{linha.name}} {{linha.value}}

Valor de saída:

{{saida2}} 
<ng-grid input="entrada2" output="saida2" var="linha" title="Listagem com paginação e seleção" rows="10">
  <column caption="ID" width="100px">{{linha.id}}</column>
  <column caption="Name" order="name" filter="name" width="200px">{{linha.name}}</column>
  <column caption="Value" order="value">{{linha.value}}</column>
</ng-grid>
